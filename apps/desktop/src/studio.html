欢迎引导

从零开始配置 BranchWhisper

这版先做成小白也能看懂的桌面应用：不装本地环境也能先跑通 API，再创建微信 Bot、设置人格、保存对话数据，最后按需要进入本地完整部署。

API 轻量部署 推荐新手先走这条，不需要 WSL、CUDA、conda、本地模型。
本地完整部署 适合已经准备本地模型、语音链路和微信桥接的人。
